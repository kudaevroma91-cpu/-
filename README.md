# Папка для сайтов

Одностраничное портфолио со scroll-driven видео, двумя кейсами и формой заявки.

## Локальный просмотр

Откройте `index.html` напрямую или запустите простой статический сервер:

```bash
python -m http.server 4173
```

После этого сайт будет доступен по адресу `http://127.0.0.1:4173/`.

## Файлы

- `index.html` — структура лендинга
- `styles.css` — оформление и адаптивность
- `script.js` — управление видео при прокрутке и форма
- `folder-scroll.mp4` — основная анимация папки
