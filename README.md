# Папка для сайтов

Одностраничный сайт со scroll-driven видео, кратким описанием услуг и прямой ссылкой на Telegram

## Локальный просмотр

Откройте `index.html` напрямую или запустите простой статический сервер:

```bash
python -m http.server 4173
```

После этого сайт будет доступен по адресу `http://127.0.0.1:4173/`.

## Файлы

- `index.html` — структура лендинга
- `styles.css` — оформление и адаптивность
- `script.js` — управление видео при прокрутке и интерфейсные эффекты
- `folder-scroll-web.mp4` — веб-версия основной анимации папки
- `sites-motion-web.mp4` — веб-версия чёрно-белой анимации экранов сайта
